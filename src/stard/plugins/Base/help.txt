General Commands:
!core - Gives you a ship core, thrusters, and power
!help <topic> - display help information on a topic
!limits - Print out ship size limits for this server.
!motd - Prints out the server message of the day.
!rules - Prints the server rules.
!stuck - (Astronaut Only) teleport 1 sector away.
!help admin - Display Admin only commands
